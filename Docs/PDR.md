# Project Design Record (PDR) - Quantum Step Creations

## Overview
Quantum Step Creations is a premium, high-performance web development studio specializing in modern, glassmorphic designs and powerful free financial tools for the Indian market.

## Architecture
- **Framework**: Static HTML/JS with Tailwind CSS (via CDN or local script).
- **Styling**: Vanilla CSS + Tailwind components.
- **Interactions**: GSAP for staggered animations, ScrollTrigger for scroll-based effects.
- **Tools Architecture**: State-based architecture with vanilla JavaScript.

## Design Principles
- **Aesthetic**: Hyper-Glass Luxury (deep translucency, vibrant gradients, 3D floating effects).
- **Typography**: Inter / Display fonts with high readability.
- **Color Palette**: 
  - Indigo (#6366f1)
  - Emerald (#10b981)
  - Pink/Highlight (#ec4899)
- **Icons**: SVG ONLY. **NO EMOJIS, WINGDINGS, or non-standard symbols in the UI.**
- **Visuals**: High-fidelity photo-realistic assets or high-end 3D renders.

## Governance & Guardrails
- **Nomenclature**: 
  - "Our Expertise" -> "Our Services"
  - "View our work" -> "Our Portfolio"
- **SEO**: Meta tags and descriptive indexing required for all pages.
- **Consultation**: "Free Consultation" must be a primary call-to-action on the landing page.
